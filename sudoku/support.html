How to Play

Sudoku is a logic-based number puzzle. The objective is to fill a 9×9 grid with digits so that:

  • Each column contains all digits from 1 to 9
  • Each row contains all digits from 1 to 9
  • Each 3×3 box contains all digits from 1 to 9

Game Controls

Selecting a Cell

Tap any empty cell to select it. Tap outside the grid to deselect.

Entering Numbers

Select a cell and tap a number from the number pad at the bottom.

Notes Mode

Tap the pencil icon to enable notes mode. Use this to mark possible numbers for a cell.

Erase

Select a cell and tap the trash icon to clear its value or notes.

Undo

Tap the undo button to revert your last action.

Hints

You have 3 hints per game. Select an empty cell and tap the hint button to reveal the correct number.

Difficulty Levels

Easy

Perfect for beginners. Requires only basic single-candidate techniques.

Medium

Hard

Advanced difficulty. Requires triples and pointing techniques.

Expert

Master level. Requires X-Wing and advanced logical techniques.

Frequently Asked Questions

How do I save my game?

Your game is automatically saved every 10 seconds and after each action. You can close the app and continue later.

What happens if I make 3 mistakes?

The game ends after 3 mistakes. You'll need to start a new puzzle.

Can I prevent overwriting correct numbers?

Yes! The app automatically protects cells with correct numbers - you cannot overwrite them.

How do notes work?

Notes help you track possible numbers for a cell. The app automatically removes invalid notes when you place numbers.

Are puzzles guaranteed to be solvable without guessing?

Yes! All puzzles are generated using logical techniques only. You never need to guess.

Как играть

Судоку - это логическая головоломка с числами. Цель - заполнить сетку 9×9 цифрами так, чтобы:

  • Каждый столбец содержал все цифры от 1 до 9
  • Каждая строка содержала все цифры от 1 до 9
  • Каждый квадрат 3×3 содержал все цифры от 1 до 9

Управление игрой

Выбор ячейки

Нажмите на любую пустую ячейку, чтобы выбрать её. Нажмите вне сетки, чтобы снять выделение.

Ввод чисел

Выберите ячейку и нажмите на число на цифровой панели внизу.

Режим заметок

Нажмите на иконку карандаша, чтобы включить режим заметок. Используйте его для отметки возможных чисел.

Стереть

Выберите ячейку и нажмите на иконку корзины, чтобы очистить значение или заметки.

Отмена

Нажмите кнопку отмены, чтобы отменить последнее действие.

Подсказки

У вас есть 3 подсказки на игру. Выберите пустую ячейку и нажмите кнопку подсказки, чтобы увидеть правильное число.

Часто задаваемые вопросы

Как сохранить игру?

Ваша игра автоматически сохраняется каждые 10 секунд и после каждого действия. Вы можете закрыть приложение и продолжить позже.

Что происходит при 3 ошибках?

Игра заканчивается после 3 ошибок. Вам нужно будет начать новую головоломку.

Можно ли предотвратить перезапись правильных чисел?

Да! Приложение автоматически защищает ячейки с правильными числами - вы не сможете их перезаписать.

Как работают заметки?

Заметки помогают отслеживать возможные числа для ячейки. Приложение автоматически удаляет недопустимые заметки при размещении чисел.

Гарантируется ли решаемость головоломок без угадывания?

Да! Все головоломки генерируются с использованием только логических техник. Вам никогда не нужно угадывать.
